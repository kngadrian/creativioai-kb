## Quick Tips for Success

### 1. Start with Quality Inputs
- Well-lit product photos (1000px+ resolution)
- Clean backgrounds (or use AI background removal)
- Sharp focus and accurate colors

### 2. Use Detailed Prompts
- Be specific: "Modern living room with morning sunlight"
- Include mood: "Minimal, peaceful aesthetic"
- Reference style: "Magazine-quality product photography"

### 3. Leverage Brand Kit
- Set up once, use everywhere
- Maintain consistency across all projects
- Save time on repetitive tasks

### 4. Batch Create & Test
- Generate multiple variations at once
- A/B test different styles and compositions
- Iterate based on performance data

### 5. Follow Platform Guidelines
- Match aspect ratios to target platforms
- Keep mobile viewing in mind (large text)
- Optimize file sizes for web
