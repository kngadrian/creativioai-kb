Brand Kit

Purpose: Define your brand’s visual identity so AI generations stay consistent (tone, colors, logo/watermark, typography).

Where to find it

  • Left sidebar → Brand Kit
  • URL: /brand

Sections observed

Brand Identity

  • Brand Name
  • Website
  • Brand Description / Voice — personality, tone, target audience (used by AI)

Logo & Watermark

  • Upload Logo (PNG/SVG) — recommends transparent background
  • Default Logo Position — choose placement on generated images
  • Logo Size — small / normal / large

Color Palette

  • Multiple color entries (hex fields) + Add Color

Typography

Typography settings section is present (details to be documented as we expand it).

Saving

Use Save Changes to apply Brand Kit settings.
