Animation Studio

Purpose: Turn images (or text prompts) into short video animations using different generation modes and quality settings.

Where to find it

  • Left sidebar → Animation Studio
  • URL: /animation

Modes (as shown in UI)

  • No Image — Text to Video
  • Static Image — Image to Video
  • First & Last — Transition
  • 3 Images — Mix / Style Ref
  • Extend Video — Longer clips

Core workflow (Image to Video example)

  1. Upload a Source Image (Choose File).
  2. Choose Model Quality:
    • Fast (150 credits)
    • Quality (400 credits)
  3. Pick Video Style (dropdown/combobox).
  4. Write an Animation Prompt (example placeholder: “Panning wide shot of the product in sunshine”).
  5. (Optional) Add a Negative Prompt (example: “blurry, distorted”).
  6. Choose Aspect Ratio (e.g., 16:9 or 9:16) and Resolution.
  7. Click Generate Video (disabled until required inputs are filled).

Tip shown in product

Video generation is compute-intensive. Start with Fast mode to test, then switch to Quality for final renders.
