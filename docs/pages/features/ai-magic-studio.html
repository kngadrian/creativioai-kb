AI Magic Studio

Purpose: A canvas-style editor where you can combine images, mask areas, add/remove text, remove backgrounds, upscale, and generate a final composition with an AI prompt.

Where to find it

  • Left sidebar → AI Magic Studio
  • URL: /magic-studio

Editor toolbar (hotkeys shown in UI)

  • Pan (H)
  • Select (V)
  • Mask Brush (B) — paint masks/regions
  • Add Text (T)
  • Remove Text (R)
  • Eraser (E)
  • Remove BG (Q)
  • Undo / Redo
  • Upscale (U)
  • Insert Image

How it works (typical flow)

  1. Drag images onto the canvas (UI hint: “Drag images here or press Ctrl+U”).
  2. Pick an aspect ratio (examples seen: 1:1 Square, 16:9 Landscape, 9:16 Portrait, 4:3, 3:4).
  3. (Optional) Use tools like Remove BG, mask brush, add/remove text, and insert images.
  4. Write a composition prompt (textbox: “Describe the final composition. All images on canvas will be included…”).
  5. Click Generate (disabled until required inputs are provided).
