# Getting started

CreativioAI helps you generate product photoshoots, edit images with AI tools, and create animated creatives.

## Main areas (left sidebar)
- **Dashboard** — overview and announcements
- **Create Photoshoot** — generate product images (AI product photography)
- **AI Magic Studio** — canvas editor for composing images + prompts
- **AI Magic Tools** — one-off tools like background removal, upscaling, relighting, etc.
- **Animation Studio** — generate short videos (image/text → video)
- **Your Assets** — your library of generated and uploaded assets
- **Brand Kit** — brand identity for consistent outputs
- **Video Training** — tutorials
- **Knowledge Base** — in-app help
- **Agency Dashboard** — clients/team management

!!! tip "Recommended order for new users"
    1. Start with **Create Photoshoot** to generate a few strong base images.
    2. Use **AI Magic Tools** to polish.
    3. Use **Animation Studio** when you want scroll-stopping video creatives.
